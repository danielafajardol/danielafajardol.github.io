1.2: Derivative of Gaussian (DoG) Filter

Approach 1: 1. Create gaussian. I chose sigma to be 2. 2. Blurred the image by convolving it with the gaussian filter using convolve2d 3. Convolved the blured image with Dx and Dy to get the partial derivatives 4. Took the gradient magnitude of the image (same equation as before) 5. Determined threshold (same as before) 6. Created binary edge image #Blurred image blurred_im = convolve2d(im_g, g_filter, mode='same', boundary='wrap') #Convolve image b_grad_x = convolve2d(blurred_im, D_x, mode='same', boundary='wrap') b_grad_y = convolve2d(blurred_im, D_y, mode='same', boundary='wrap') #Compute the gradient magnitude image b_grad_magnitude = np.sqrt(b_grad_x**2 + b_grad_y**2) #Determine threshold b_threshold = np.max(b_grad_magnitude) * 0.36 # Create a binary edge image b_edge_image = b_grad_magnitude > b_threshold
